Quick Setup 5 steps
1
Create Alpaca Account
Sign up free at alpaca.markets. Enable paper trading (no real money needed to test). UK residents fully supported.
2
Get API Keys
In Alpaca dashboard → API Keys → Generate. Copy your API Key ID and Secret Key.
3
Fork GitHub Repo & Add Secrets
In your repo → Settings → Secrets → Actions. Add these secrets:
ALPACA_API_KEY
ALPACA_SECRET_KEY
ALPACA_BASE_URL = https://paper-api.alpaca.markets
4
Enable GitHub Actions
Go to Actions tab → Enable workflows. Bot runs automatically every 10 minutes during US market hours.
5
Go Live
After testing on paper, fund your Alpaca account and change the base URL secret to:
ALPACA_BASE_URL = https://api.alpaca.markets
